GitHub Access Token (one-time setup)

This token lets the admin panel upload images directly to your GitHub repo. It is stored only in your browser — never sent anywhere else.

How to create a GitHub token →
  1. Go to github.com/settings/tokens
  2. Click "Generate new token (classic)"
  3. Note: bed-linen-admin
  4. Expiration: 90 days or "No expiration"
  5. Scopes: tick repo (full control)
  6. Click "Generate token" → copy it
  7. Paste the token above and click Save
